Set up pip#

pip is the main package installer and controler for Pyhon packages available through PyPi (the vast majority of them are).

1. Why Use pip?#

pip is the package installer for Python. You can use it to install and manage packages and dependencies for your Python projects.

2. Install pip`#

`pip is included by default with Python installations since version 3.4. If you have an earlier version of Python, you can download the installation script from the official Python website.

2.1 macOS (using Homebrew)#

  1. Install Homebrew if you haven‚Äôt already.

  2. Run the following command in your terminal: brew install python

2.2 Linux#

  1. Run the following command in your terminal: sudo apt-get install python3-pip (for Ubuntu-based distributions) or sudo yum install python3-pip (for RPM-based distributions)

3. Verify pip Installation#

  1. Open a terminal or command prompt.

  2. Type pip --version and press Enter.

  3. If `pip is installed correctly, you should see the version number.

4. Commonly used commands#

  1. Upgrade pip: pip install --upgrade pip

  2. Install package: pip install <package-name>

  3. Upgrade package: pip install --upgrade <package-name>

  4. Install specific version of package: pip install <package-name>==0.0.0>

  5. Uninstall package: pip uninstall <package-name>>

  6. Show package information: pip show <package-name>
