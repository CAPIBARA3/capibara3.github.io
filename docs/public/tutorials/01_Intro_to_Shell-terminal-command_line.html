1. What is the Shell?#

The shell, also known as the command line, terminal, or command prompt, is an interface that allows you to interact directly with your computer‚Äôs operating system using text-based commands. It may look intimidating at first, but it‚Äôs a powerful tool that gives you control over your system.

In a terminal, you type commands to:

  • Navigate the file system

  • Create, move, and delete files and directories

  • Install software packages

  • And much more!

2. Opening the Terminal#

If you‚Äôre using macOS or Linux, the terminal should already be installed. You can open it by:

  • macOS: Press Cmd + Space, type ‚ÄúTerminal‚Äù, and hit Enter.

  • Linux: Press Ctrl + Alt + T or search for ‚ÄúTerminal‚Äù in your application menu.

3. Basic Terminal Commands#

Here are some fundamental commands to get started. Try typing these into your terminal:

Navigating Directories#

  • pwd ‚Äî Print Working Directory: Shows the current directory you‚Äôre in.

    pwd

Output example: /home/username

  • ls ‚Äî List: Lists the files and directories in your current directory.

    ls

Output example: Documents  Downloads  Pictures  Projects

  • cd ‚Äî Change Directory: Changes the current directory.

    cd Documents

use cd ..to go up one level in the directory tree.

Managing Files and Directories#

  • mkdir ‚Äî Make Directory: Creates a new directory.

    mkdir my_new_directory

  • touch - Create Empty File: Creates an empty file.

    touch myfile.txt

  • rm ‚Äî Remove: Deletes a file.

    rm myfile.txt

  • rmdir ‚Äî Remove Directory: Deletes an empty directory.

    rmdir my_new_directory

Viewing and Editing Files#

  • cat ‚Äî Concatenate: Displays the content of a file.

    cat myfile.txt

  • nano ‚Äî Text Editor: A simple text editor to edit files.

    nano myfile.txt

Press Ctrl + X to exit. Press Y to confirm saving changes.

4. Using the man Command#

Many commands in the terminal have built-in help. The man` command is used to open the manual (help documentation) for a command. For example:

ls

This will show detailed information about the ls command and its options.

5. Understanding the File System#

In Unix-like systems, files and directories are organized in a hierarchical structure. Here‚Äôs a basic rundown of the common directories you may encounter:

  • / ‚Äî The root directory, the top of the file system.

  • /home/username/ ‚Äî Your personal home directory.

  • /bin/ ‚Äî Essential system binaries (programs).

  • /usr/ ‚Äî User utilities and applications.

6. Permissions in the Terminal#

You might see some commands fail or require elevated privileges. This is because files and directories in Unix-like systems have permissions. For example:

  • chmod ‚Äî Changes file permissions.

    chmod +x myscript.sh

  • sudo ‚Äî Execute commands as the superuser (administrator).

Note: You‚Äôll need to enter your password when using sudo.

7. A Few More Useful Commands#

  • clear ‚Äî Clears the terminal screen.

    clear

  • exit - Closes the terminal window.

8. Practice!#

The best way to get comfortable with the terminal is through practice. Try using the commands above to navigate, create files, and explore your system. You can always use man <command> to get more information about any command.
