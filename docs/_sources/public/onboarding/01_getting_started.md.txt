# 01 Getting Started

## Goal
Once you have submitted the application and received an invitation to join the CAPIBARA Collaboration per email, you will need to complete the following steps to become a full member of the collaboration.

## Prerequisites
- Have submitted the application form
- Fulfill the membership requirements (see [membership agreement](../governance/1_membership.md))
- Have received invitation to join the collaboration via email from the official collaboration email.

## Steps

### 1. Introduction
Congratulations on being accepted to join the CAPIBARA Collaboration! This tutorial will guide you through the essential first steps to become a full member and set up access to our collaboration's tools, repositories, and communication channels. You're about to join a community of student researchers and developers focused on open science and collaborative software development.

### 2. Setup
Before starting this process you will need to have created a GitHub account. If you have one, you just need to remember the login credentials (for you to log in). If not you can find a very detailed guide [here](https://docs.github.com/en/get-started/start-your-journey/creating-an-account-on-github). GitHub is our main information hub all of our codes, documentation and files are stored in our organisation ([github.com/capibara3](https://github.com/capibara3)). Decision-making also takes place there through asynchronous communication within issues, pull requests, and discussions. You will learn the "GitHub workflow" once you reach the next tutorials.

### 3. Execution
In order to finalise the joining process, you need to enter the CAPIBARA GitHub organisation. For that matter, you will need to submit the following information by replying to the email with the invitation to join the collaboration.

1. **GitHub username**: so that we can invite you to join the organisation on GitHub
2. For our team website ([capibara3.github.io/team](https://capibara3.github.io/team)) you may want to provide the following information:
* Full Name
* Short statement
* Description
* Profile Picture
* Links (LinkedIn, YouTube, website, email, get inspired by other profiles!)
* Short presentation video (30 sec max)

> _Providing this data is optional, and you may also request not to have a public profile or to delete your data anytime. By sending the data, you agree to it being treated as outlined here. It will be hosted on GitHub Pages and be publicly available on the CAPIBARA Collaborationâ€™s website._

### 4. Verification
You can verify this step is complete when:
- You have received confirmation of your GitHub organization membership
- You can access [github.com/capibara3](https://github.com/capibara3) and view private repositories
- Your profile has been added to the team website (if you submitted the optional information)
- You have reviewed the Code of Conduct and Contributing guidelines

## Next Steps
Once you have joined the CAPIBARA GitHub organisation, you can start contributing! Plus, you also have access to all of our private files. Make sure to always comply with the code of conduct and contribution expectations, both collaboration-wide and repository-specific guidelines. You will learn about GitHub in a future tutorial.
